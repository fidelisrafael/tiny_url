tiny_url by Rodrigo Vieira

A gem for tinifying urls

Installation

Install tiny_url with RubyGems:

 $ gem install tiny_url

Usage

From the command-line:

 $ tiny_url http://accidentalchinesehipsters.tumblr.com/
This command tinifies the url for you, copies it to the clipboard and opens the website in your browser.

From anywhere else:


      require "tiny_url"
      
      TinyUrl.silently_tinify "http://thereifixedit.failblog.org/"
